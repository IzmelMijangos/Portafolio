Sobre Mí

Soy Izmel Torres, un Ingeniero Mecatrónico y desarrollador Full Stack, me especializo en convertir desafíos complejos en sistemas eficientes y elegantes. Mi trayectoria se define por una constante evolución profesional. Manteniendo un conocimiento al día con las últimas tendencias tecnológicas, me gusta aportar a los proyectos ideas frescas y métodos probados que impulsan la innovación y eficacia operativa. Cuento con un historial de proyectos que hablan de compromiso y habilidad técnica, busco oportunidades que me permitan seguir creciendo profesionalmente y dejar una marca positiva en el sector tecnológico. Si buscas a un profesional que no solo entiende las tendencias actuales, sino que también contribuye a darles forma, te invito a conocer más sobre mi experiencia.

Competencias Técnicas

Desarrollo de Software

Profundo conocimiento en la creación de aplicaciones robustas y de alto rendimiento. Amplia experiencia en el ciclo completo de desarrollo de software desde la concepción hasta la implementación y el mantenimiento. Backend: Diseño y desarrollo de arquitecturas de servidor, API y microservicios utilizando Django, Flask, Spring Boot, así como el uso de MySQL y Postgresql para el manejo de los datos. Frontend: Creación de interfaces de usuario interactivas y responsivas con Vue.js, React, JavaScript.

Infraestructura y Operaciones (DevOps)

Aplicación de principios de DevOps para mejorar la colaboración entre desarrollo y operaciones, incrementando la eficiencia en la entrega de software y la escalabilidad de infraestructura. Automatización de CI/CD: Implementación de pipelines de integración continua y entrega/deploy continua con Jenkins, GitHub Actions y GitLab CI para el control de versiones y la automatización del flujo de trabajo de software. Orquestación de Contenedores: Manejo de contenedores y microservicios en entornos de producción utilizando Docker, y Amazon Web Services (AWS) para la gestión y escalado eficiente de aplicaciones.

Cloud Computing y Servicios AWS

Gestión y optimización de servicios en la nube para escalabilidad, seguridad y coste efectividad, especializado en el ecosistema de AWS. Computación en la Nube: Configuración y mantenimiento de sistemas escalables y tolerantes a fallos en AWS utilizando Elastic Compute Cloud (EC2), Elastic Kubernetes Service (EKS), y Elastic Beanstalk para despliegues rápidos y gestión simplificada. Servicios AWS Específicos: Experiencia con Amazon Simple Storage Service (S3) para almacenamiento de objetos, AWS Lambda para computación sin servidor, Amazon RDS para bases de datos relacionales, y AWS CloudFormation para la infraestructura como código, diseñando soluciones seguras y optimizadas para la nube.

Arquitectura de Software

Como Arquitecto de Software, especializado en diseñar soluciones escalables y sostenibles, me enfoco en la creación de arquitecturas resilientes que respalden los objetivos de negocio y la innovación continua. Mis competencias abarcan:

  • Definición de arquitecturas de microservicios para facilitar el despliegue ágil y la mantenibilidad.
  • Implementación de patrones de diseño y arquitectónicos que aseguran la cohesión y la baja acoplamiento.
  • Evaluación y adopción de tecnologías emergentes para mantener las soluciones tecnológicas alineadas con las mejores prácticas del sector.
  • Colaboración estrecha con equipos de desarrollo para garantizar la aplicación efectiva de los principios arquitectónicos.
  • Gestión de riesgos técnicos y aseguramiento de la calidad a través de revisiones de código y pruebas automatizadas.

Mi enfoque pragmático y centrado en el usuario final garantiza que las arquitecturas desarrolladas no solo son técnicamente sólidas, sino también alineadas con las necesidades comerciales y la experiencia del usuario.
